# Dr. Chhabi Adhikari — D-School System

Official website for **Dr. Chhabi Adhikari**, a pioneer NLP (Neuro-Linguistic Programming) trainer and life coach based in Nepal. The **D-School System** is an educational platform dedicated to mindset transformation, personal excellence, and professional coaching certification.

## 🌟 Project Overview

This project is a comprehensive web portal that showcases the professional work, training programs, and resources offered by Dr. Chhabi Adhikari. It serves as:
- A landing page for live NLP workshops and certifications.
- An e-learning hub for online courses and audio programs.
- A resource center for personal and business transformation through NLP.

## 🚀 Key Features

- **Workshop Registration**: Detailed information and schedules for upcoming live workshops (NLP Practitioner, Master Practitioner, TTCLC).
- **Online Academy**: Access to digital programs like Student Memory Mastery, Wellness Mastery, and Money Mastery.
- **Subconscious Mind Programming**: A collection of audio and video programs designed for subconscious re-imprinting.
- **Personal & Corporate Coaching**: Specialized modules for one-on-one counseling and corporate in-house training.
- **Resource Blog**: Insights and articles on personal mastery, business growth, and psychology.
- **Multilingual/Regional Focus**: Tailored for the Nepalese and South Asian audience.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+).
- **Backend**: PHP (Dynamic Templating & Form Handling).
- **Database**: MySQL (for lead generation and contact management).
- **Icons & Typography**: FontAwesome, Google Fonts (Inter, Outfit).
- **External Integration**: YouTube (Video resources), PWA (Progressive Web App support).

## 📁 Project Structure

- `index.php`: The main entry point featuring the hero slider and core value proposition.
- `includes/`: Reusable components (Header, Footer).
- `chhabi sir/`: Brand-specific image assets and media.
- `css/` & `js/`: Core styling and interactive logic.
- `ramverma-COMPLETE-PRD.md`: The original Product Requirements Document used as a reference for site fidelity and structure.

## 📖 Installation

To run this project locally:
1. Ensure you have a PHP environment installed (e.g., XAMPP, WAMP, or Apache).
2. Clone this repository into your `htdocs` or `www` directory.
3. Import the database (if provided) and configure `connect.php`.
4. Run the project through your local server (e.g., `http://localhost/d-sch`).
